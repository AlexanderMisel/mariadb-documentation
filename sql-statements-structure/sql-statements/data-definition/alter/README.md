# ALTER

This category is for documentation on the various ALTER statements.

- [ALTER TABLE](/sql-statements-structure/sql-statements/data-definition/alter/alter-table/) — Modify a table's definition.
- [ALTER DATABASE](/sql-statements-structure/sql-statements/data-definition/alter/alter-database/) — Change the overall characteristics of a database.
- [ALTER EVENT](/programming-customizing-mariadb/triggers-events/event-scheduler/alter-event/) — Change an existing event.
- [ALTER FUNCTION](/sql-statements-structure/sql-statements/data-definition/alter/alter-function/) — Change the characteristics of a stored function.
- [ALTER LOGFILE GROUP](/sql-statements-structure/sql-statements/data-definition/alter/alter-logfile-group/) — Only useful with MySQL Cluster, and has no effect in MariaDB.
- [ALTER PROCEDURE](/programming-customizing-mariadb/stored-routines/stored-procedures/alter-procedure/) — Change stored procedure characteristics.
- [ALTER SEQUENCE](/sql-statements-structure/sequences/alter-sequence/) — Change options for a SEQUENCE.
- [ALTER SERVER](/sql-statements-structure/sql-statements/data-definition/alter/alter-server/) — Updates mysql.servers table.
- [ALTER TABLESPACE](/sql-statements-structure/sql-statements/data-definition/alter/alter-tablespace/) — ALTER TABLESPACE is not available in MariaDB.
- [ALTER USER](/sql-statements-structure/sql-statements/account-management-sql-commands/alter-user/) — Modify an existing MariaDB account.
- [ALTER VIEW](/programming-customizing-mariadb/views/alter-view/) — Change a view definition.
