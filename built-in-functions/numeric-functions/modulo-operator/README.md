# Modulo Operator (%)

## Syntax

```sql
N % M
```

## Description

Modulo operator. Returns the remainder of `N` divided by `M`. See also [MOD](/built-in-functions/numeric-functions/mod).

## Examples

```sql
SELECT 1042 % 50;
+-----------+
| 1042 % 50 |
+-----------+
|        42 |
+-----------+
```
