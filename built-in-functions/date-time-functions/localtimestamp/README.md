# LOCALTIMESTAMP

## Syntax

```sql
LOCALTIMESTAMP
LOCALTIMESTAMP([precision])
```

## Description

`LOCALTIMESTAMP` and `LOCALTIMESTAMP()` are synonyms for [NOW()](/built-in-functions/date-time-functions/now/).
