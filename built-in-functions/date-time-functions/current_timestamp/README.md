# CURRENT_TIMESTAMP

## Syntax

```sql
CURRENT_TIMESTAMP
CURRENT_TIMESTAMP([precision])
```

## Description

`CURRENT_TIMESTAMP` and `CURRENT_TIMESTAMP()` are synonyms for [NOW()](/built-in-functions/date-time-functions/now).
