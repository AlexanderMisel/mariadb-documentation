# CURRENT_TIME

## Syntax

```sql
CURRENT_TIME
CURRENT_TIME([precision])
```

## Description

`CURRENT_TIME` and `CURRENT_TIME()` are synonyms for [CURTIME()](/built-in-functions/date-time-functions/curtime/).
